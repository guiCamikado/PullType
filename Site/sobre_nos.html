Esse site foi criado por Guilherme Andrade Camikado como um projeto para aprender e me aperfeiçoar em javaScript, HTML, CSS, Banco de dados além de me aperfeiçoar e conhecer e criar páginas web.

O objetivo principal é o de criar uma espécie de página que possibilite a criação de guias com layouts facilmente editaveis e organizados de forma de facil leitura, criação e compreenção utilizando-se de javaScript, CSS e HTML além de banco de dados para o armazenamento de guias que possam ser criados por usuários e acessados facilmente.

Como inspiração para o site visei páginas como MobasFire e WikiHow além de aplicações de notação como Notion e Anki flashCards
